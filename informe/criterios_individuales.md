# Criterios asumidos para el desarrollo

- Empresa ficticia elegida: AndesMarket S.A.C.
- Modalidad de desarrollo: individual.
- Herramientas principales: Python, Jupyter Notebook y Power BI Desktop.
- Power BI se trabajara en una maquina virtual porque el entorno actual es Linux.
- Los notebooks de Python deben dejar archivos CSV exportados para que Power BI pueda consumirlos directamente.
- La bitacora queda sin fechas concretas para que el estudiante la ajuste antes de la entrega.
