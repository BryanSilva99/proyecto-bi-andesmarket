# Instrucciones para exportar el informe final

El contenido base esta en:

```text
informe/Informe_PG_AndesMarket.md
```

## Opcion recomendada: Word o Google Docs

1. Abrir `Informe_PG_AndesMarket.md`.
2. Copiar el contenido a Word o Google Docs.
3. Aplicar formato institucional:
   - Fuente Arial.
   - Documento en blanco y negro.
   - Encabezado: `UNMSM - FISI - E.P. Ingenieria de Software`, asignatura y Proyecto Grupal.
   - Pie de pagina: `Prof. Juan Gamarra Moreno` y numeracion `Pag X de Y`.
4. Insertar capturas del archivo `powerbi/AndesMarket.pbix`:
   - Modelo de relaciones.
   - Resumen ejecutivo.
   - Riesgo de abandono.
   - Segmentacion de clientes.
   - Canasta de mercado.
   - Pronostico de demanda.
5. Insertar graficos relevantes desde `informe/img/`.
6. Revisar redaccion con palabras propias.
7. Exportar como:

```text
informe/Informe_PG_AndesMarket.pdf
```

## Capturas minimas sugeridas

- Modelo estrella en Power BI.
- Pagina Resumen Ejecutivo.
- Pagina Categorias y Productos.
- Pagina Riesgo de abandono.
- Pagina Segmentacion de clientes.
- Pagina Canasta de mercado.
- Pagina Pronostico de demanda.

Antes de exportar, revisar:
